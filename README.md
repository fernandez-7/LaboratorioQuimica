# 🧪 Laboratorio de Química Virtual (Digital Twin) - I.E.P Latino

¡Bienvenido al repositorio oficial del **Laboratorio de Química Virtual**! Este proyecto es un Gemelo Digital (Digital Twin) interactivo desarrollado en Unity, diseñado para transformar la educación científica mediante la simulación inmersiva y en tiempo real de un entorno de laboratorio escolar.

---

## 🎯 Problema que Resuelve

Muchas instituciones educativas enfrentan una brecha crítica entre la teoría y la práctica en la enseñanza de las ciencias básicas debido a:
* **Falta de infraestructura física:** Carencia de espacios acondicionados o laboratorios equipados en los colegios.
* **Costos elevados:** Presupuesto limitado para la compra de reactivos, cristalería e instrumentos especializados.
* **Riesgos de seguridad:** Peligro inherente al manejo incorrecto de compuestos químicos, quemaduras o inhalación de sustancias por parte de los estudiantes.

Este **Digital Twin de Laboratorio** resuelve esta problemática permitiendo el acceso seguro a la experimentación científica. Los alumnos interactúan con materiales y procesos químicos de forma **totalmente segura, económica, reutilizable e ilimitada**, eliminando las barreras de espacio y riesgos de accidentes físicos.

---

## 🎮 Controles del Simulador

El sistema incluye una lógica de cámaras dinámicas (Primera y Tercera Persona). Puedes operar el entorno con los siguientes comandos:

| Tecla / Acción | Función |
| :--- | :--- |
| **W, A, S, D** / Flechas | Movimiento del personaje (Avanzar, retroceder, izquierda, derecha). |
| **Mouse (Movimiento)** | Rotación de la cámara y orientación del personaje. |
| **Shift Izquierdo (Mantener)** | Correr (Incrementa la velocidad de desplazamiento). |
| **Tecla Q** | **Cambiar de Vista:** Alterna entre primera persona (FP) y tercera persona (TP). |
| **Tecla Escape** | Liberar el cursor del mouse de la ventana del juego. |

## 🛠️ Tecnologías Utilizadas

* **Motor de Desarrollo:** Unity 2022.3.62f3 LTS.
* **Lenguaje de Programación:** C# (.NET).
* **Gráficos y Animación:** Mixamo (Rigging y animaciones humanoides integradas mediante Animator Controllers).
* **Interfaz de Usuario (UI):** TextMeshPro para textos vectoriales en alta definición.
* **Control de Versiones:** Git & GitHub.

---

## 💻 Requisitos del Sistema

### 1. Software Base Requerido
* **Unity Hub:** Versión más reciente instalada.
* **Editor de Unity:** Versión exacta `2022.3.62f3 LTS`. Debe contar con el módulo de exportación de tu sistema operativo (ej. *Windows Build Support*).
* **Git:** Para clonar correctamente el repositorio.

### 2. Hardware Recomendado
* **Procesador:** Intel Core i5 / AMD Ryzen 5 (4 núcleos o superior).
* **Memoria RAM:** 8 GB como mínimo (16 GB recomendado para fluidez en el editor).
* **Tarjeta Gráfica:** NVIDIA GTX 1050 Ti / AMD RX 570 o superior (compatible con DirectX 11/12).
