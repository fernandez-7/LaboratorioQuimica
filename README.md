# 🧪 Digital Twin de Laboratorio

## 🎯 Problema que Resuelve
Muchas instituciones educativas enfrentan una brecha crítica entre la teoría y la práctica debido a la falta de infraestructura física, laboratorios equipados y presupuesto para reactivos. Esto limita la enseñanza de las ciencias a un modelo puramente expositivo, dificultando la comprensión de conceptos complejos.

Este **Digital Twin de Laboratorio** resuelve esta problemática al permitir el acceso a la experimentación en un entorno virtual interactivo que replica un laboratorio de Química real, permitiendo a los estudiantes interactuar con materiales y procesos químicos de forma segura, económica y sin límites de espacio.

## 🛠️ Tecnologías Usadas
* **Motor de desarrollo:** Unity 2022.3 LTS.
* **Gestión del proyecto:** Unity Hub.
* **Animación:** Mixamo (Rigging y animaciones humanoides).

## 💻 Requisitos de Instalación

### 1. Software Base
* **Unity Hub:** Versión más reciente.
* **Editor de Unity:** Versión `2022.3.62f3 LTS`. Es fundamental contar con los módulos de exportación instalados (ej. Android Build Support si el objetivo es VR o móvil).
* **Git:** Recomendado para clonar y gestionar las versiones del repositorio.

### 2. Hardware Recomendado
* **Procesador:** Intel Core i5 / AMD Ryzen 5 o superior.
* **Memoria RAM:** 8 GB mínimo (16 GB recomendado para fluidez en el editor).
* **Tarjeta Gráfica:** NVIDIA RTX 3050 o equivalente (compatible con DirectX 11/12).
